## Judgment calls to double-check

- **"Agnješka"** — Serbian phonetic rendering of Polish "Agnieszka" (pronounced ag-NYESH-ka). Common Serbian media transliteration, but flag if you'd prefer a different spelling.
- **"Vjeslav"** — Serbian phonetic rendering of Polish "Wiesław" (the Polish ł doesn't exist in Serbian). Alternative: keep the Polish spelling as-is for proper nouns.
- **"Neša"** vs **"Nesa"** — used "Neša" (nickname form, matches how it's pronounced) consistently for the character Nesa; confirm this is the spelling you want going forward.
- **"a" vocab entry** — added as a course word (contrastive "and/but", e.g. "Ti si ovde, a on nije") even though it wasn't explicitly listed in section1.json's vocabFocus; it's a high-frequency function word that shows up naturally once sentences get longer.
- **"student" vocab entry** — same situation: a transparent EN/PL/SR cognate, added because it made a natural fill_blank sentence; not in the original vocabFocus list.

## Judgment calls — l4/l5 (2026-07-09 repetition-lesson pass)

- **"student" applied to Lidija and Neša (both female)** — u1-l4-c02/c09 and u1-l5-c07 reuse the existing "student" noun (already flagged male-only in the original vocab table) for female characters without switching to a feminine form (e.g. "studentkinja"). Kept for scope discipline — introducing a new gendered noun form would be new vocab, which this repetition pass was explicitly told not to do — but flag if this reads as wrong to a native speaker; may be worth a real fix in a future unit that properly introduces gendered nouns.
- **"nisi" (u1-l4-c03, "a ti nisi")** — the 2nd-person-singular negated form of "biti" wasn't literally used in l1-l3 (which used nisam/nije), but it's the same present-tense-of-biti paradigm the unit's grammarFocus already covers (singular "to be", all persons implied by "si"/"ti" already taught). Treated as within-scope conjugation, not new vocab — flag if you'd rather this wait for a unit that explicitly drills the full paradigm.
- **No new vocab or grammar was introduced in l4/l5** — every sentence recombines u1's existing 19 vocab entries and cast names in new orders/pairings. This is a pure repetition pass per Luka's feedback ("3-5 lessons for repetition... not new grammar/vocab scope creep").
