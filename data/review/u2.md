## Judgment calls worth a second look

- **"Koliko imaš pasa?" / "Koliko godina imaš?"** — genitive plural after "koliko" (how many) is idiomatic Serbian even though the answer might be a small number; this is standard usage, not a paucal-form slip.
- **"već pet godina"** ("for five years") uses genitive plural "godina" since numbers 5+ take gen.pl, unlike 2-4 which take the paucal form (compare "dve godine" elsewhere in the unit vs. "pet godina" here) — the contrast is intentional, so Dominika sees both patterns.
- **Numbers governing case**: dva/dve + masc noun → looks like genitive singular ("dva psa", "dva grada"); dve/tri/četiri + fem noun → looks like nominative plural ("dve mačke", "tri mačke", "četiri knjige"). This is the real, correct Serbian pattern — flag if it reads confusingly for a first plural-forms unit; we can add a grammar-tip pill later if needed.
- **"tetke"** — Lidija and Biljana are the aunts per the recurring cast; introduced "tetka" as new vocab here since it fits naturally.
- Word order in match_meaning options mixes correct answer position (not always first) so Dominika can't pattern-match position instead of reading.
